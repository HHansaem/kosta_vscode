26일은 전국에 가끔 구름이 낀 곳이 많다. 아침 최저기온은 영하 5∼영상 3도, 낮 최고기온은 영상 4∼12도로 예보됐다. 미세먼지 농도는 전 권역이 '좋음'∼'보통' 수준으로 예상된다. 강원 영동과 경북 동해안은 대체로 흐린 가운데 일부 지역에서 새벽부터 비 또는 눈이 내려 밤까지 이어지는 곳이 있겠다.
